# vibedesign — AI Moda Görsel Üretimi

[vibedesignapp.com](https://vibedesignapp.com/) benzeri React landing page.

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:5173` adresini açın.

## Bölümler

- Hero — kampanya mesajı ve önce/sonra görselleri
- Detay Koruma — ürün detaylarının korunması
- 3 Adım — model, kıyafet, sahne
- Araçlar — Decoupe, poz/açı varyasyonları, editör
- Motion — fotoğraftan videoya
- Mekan Render — mobilya/dekorasyon sahneleri
- CTA ve footer

## Teknolojiler

- React 19 + Vite
- Tailwind CSS v4
